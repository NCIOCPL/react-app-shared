# Migration Guide

This guide covers migrating from application-local components to `@nciocpl/react-components`.

## General Steps

1. Install the package: `npm install @nciocpl/react-components`
2. Replace local component imports with library imports
3. Adjust props to match the library API
4. Remove local component files once migration is verified
